== Quais são as novidades na versão mais recente do NetApp ONTAP para AFX?

Última versão:

ONTAP 9.19.1RC1 (a partir de maio de 2026)

Novos recursos no NetApp AFX:

* Desduplicação global
* Eficiência de storage dinâmica
* Leitura antecipada avançada
* Suporte para 32 nós
* Suporte para 32PB
* 512 volumes constituintes por FlexGroup volume
